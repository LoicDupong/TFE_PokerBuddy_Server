## Tech Stack

- **Frontend:** Next.js (App Router)
- **Styling:** SCSS / Sass (no Tailwind)
- **Backend:** Express.js
- **Database:** PostgreSQL + Sequelize
- **Auth:** JWT
- **Approach:** mobile-first, front-first

- clean architecture
